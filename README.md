### 1. Clone the repository

---

### 2. Backend Setup

cd server  
npm install  

Create a `.env` file inside the server folder:

MONGO_URI=your_mongodb_connection_string  
JWT_SECRET=your_secret_key  
PORT=5000  

Start the backend server:

npm start  

---

### 3. Frontend Setup

cd client  
npm install  
npm run dev
